What is CSS?

The art of web design

CSS stands for Cascading Style Sheets. It's the language used to describe the visual presentation of web pages. While HTML provides structure, CSS adds style, controlling colors, fonts, layouts, spacing, and animations.

/* This is a CSS comment */
h1 {
    color: #3b82f6;
    font-size: 2rem;
    font-weight: bold;
}

p {
    color: #64748b;
    line-height: 1.6;
}

Think of it This Way

HTML is the skeleton of a house, and CSS is the interior design - choosing paint colors, furniture placement, and decorations.

CSS Selectors & Properties

Targeting elements with precision

CSS selectors are patterns used to select HTML elements you want to style. Once selected, you can apply various properties to change their appearance and behavior.

Common Selectors

/* Element selector */
p { color: blue; }

/* Class selector */
.highlight { background: yellow; }

/* ID selector */
#header { font-size: 24px; }

/* Multiple selectors */
h1, h2, h3 { font-family: Arial; }

Properties in Action

.card {
    background-color: #ffffff;
    border-radius: 12px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.1);
    padding: 20px;
    margin: 10px;
    transition: all 0.3s ease;
}

Understanding the Box Model

Every element is a box with layers

The CSS Box Model is fundamental to web layout. Every HTML element is essentially a rectangular box consisting of content, padding, border, and margin. Understanding this concept is crucial for controlling spacing and layout.

MARGIN
BORDER
PADDING
CONTENT

This is where your actual content lives

.box-example {
    /* Content dimensions */
    width: 200px;
    height: 100px;

    /* Padding - space inside the border */
    padding: 20px;

    /* Border - the edge of the element */
    border: 3px solid #3b82f6;

    /* Margin - space outside the border */
    margin: 15px;
}

Pro Tip

Use box-sizing: border-box to include padding and border in the element's total width and height calculation.

Modern Layouts with Flexbox

Flexible and powerful layout system

Flexbox (Flexible Box Layout) is a one-dimensional layout method that helps you arrange items in rows or columns. It's perfect for creating responsive designs and aligning elements with ease.

Flex Container Properties

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 20px;
    flex-direction: row;
}
Try Different Values:
  • justify-content: flex-start, center, flex-end, space-between
  • align-items: flex-start, center, flex-end, stretch
  • flex-direction: row, column, row-reverse, column-reverse

CSS Grid Layout

Two-dimensional layout mastery

CSS Grid Layout is a two-dimensional system that can handle both rows and columns simultaneously. It's perfect for creating complex layouts and is considered the most powerful layout system available in CSS.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 200px 100px;
    gap: 20px;
    padding: 20px;
}

.grid-item {
    background: linear-gradient(135deg, #667eea, #764ba2);
    border-radius: 12px;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-weight: bold;
}
